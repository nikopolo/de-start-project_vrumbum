Модель БД.
Разделил по таблицам повторяющие значения, это таблицы с уникальными значениями:
1. car_shop.colors
	- цвета машин;
2. car_shop.clients
	- клиенты с разделенными first_name и last_name
	- номер телефона для каждого клиента
	Создавал через временную таблицу car_shop.temp_person для отделения сокращения Jr.
3. car_shop.origins_country
	- страны
4. car_shop.brand_cars
	- бренды машин
5. car_shop.car_models
	- модели машин
6. car_shop.clients_discount
	- скидки для клиетов из таблицы car_shop.clients. У одного клиента может быть несколько скидок.
7. car_shop.cars_colors
	- промежуточная таблица для соединения многие ко многим, между car_shop.colors и car_shop.cars.

Основная таблица car_shop.cars, обеспечивающая связь с указанными таблицами.
